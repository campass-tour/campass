
## Tech Stack

- Next.js (customized, not standard)
- TypeScript
- React
- Prisma (for database/schema)
- Tailwind CSS & PostCSS
- ESLint

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```
3. Open http://localhost:3000 in your browser.
